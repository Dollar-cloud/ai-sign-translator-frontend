Sign Language Guide – one image per letter, number, and word

Put each image in this folder (assets/images/) with the EXACT filename below.
When the user clicks "A", the app loads asl-a.png (ASL mode) or isl-a.png (ISL mode).

------------------------------------------------------------
ASL MODE (prefix: asl-)
------------------------------------------------------------

Alphabet (one image per letter):
  asl-a.png, asl-b.png, asl-c.png, asl-d.png, asl-e.png, asl-f.png,
  asl-g.png, asl-h.png, asl-i.png, asl-j.png, asl-k.png, asl-l.png,
  asl-m.png, asl-n.png, asl-o.png, asl-p.png, asl-q.png, asl-r.png,
  asl-s.png, asl-t.png, asl-u.png, asl-v.png, asl-w.png, asl-x.png,
  asl-y.png, asl-z.png

Numbers (0–9 only):
  asl-0.png, asl-1.png, asl-2.png, asl-3.png, asl-4.png, asl-5.png,
  asl-6.png, asl-7.png, asl-8.png, asl-9.png

Common words (one image per word):
  asl-hi.png          (Hello)
  asl-bye.png        (Good Bye)
  asl-yes.png        (Yes)
  asl-no.png         (No)
  asl-thank-you.png  (Thanks)
  asl-welcome.png    (You're Welcome)
  asl-please.png     (Please)
  asl-sorry.png      (Sorry)

------------------------------------------------------------
ISL MODE (prefix: isl-)
------------------------------------------------------------

Same names, but with isl- instead of asl-:

Alphabet:  isl-a.png, isl-b.png, ... isl-z.png
Numbers:   isl-0.png, isl-1.png, ... isl-9.png
Words:     isl-hi.png, isl-bye.png, isl-yes.png, isl-no.png,
           isl-thank-you.png, isl-welcome.png, isl-please.png, isl-sorry.png

------------------------------------------------------------
Examples
------------------------------------------------------------
  - When user selects ASL and clicks "A"  → app loads assets/images/asl-a.png
  - When user selects ASL and clicks "Hello" → app loads assets/images/asl-hi.png
  - When user selects ISL and clicks "5"  → app loads assets/images/isl-5.png

You can add images gradually: if a file is missing, the app shows
"Photo not found. Add asl-a.png to the assets/images folder."
